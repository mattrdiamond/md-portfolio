about

Hey there, my name is Matt and I’m a Designer and Developer from Chicago. I began my career unknowingly when I discovered Microsoft Paint on Windows 3.0. Once I mastered the spray can, I knew I was ready to take my love of art and growing interest in technology to the next level. My ambition and commitment to lifelong learning has helped me succeed in a multitude of roles throughout my career including visual designer, art director, developer and project manager. I have always had a passion for building things and solving problems, and I am constantly discovering new ways to bring my ideas to life. work

Modernist.

Shop Modernist for an eclectic selection of modern and mid-century furniture, lighting, and home decor to complement your modern lifestyle. Modernist is a full-stack e-commerce application built with React, Redux, Firebase, the Stripe API, Node.js and Express.

The Loop: City Guide

Check out The Loop for the best restaurants, bars, coffee houses and attractions in the windy city! This single page React application fetches venue information from the Foursquare API and integrates Google Maps to display each venue's location.

Chatbox

ChatBox is a simple chat application built with React and Redux that allows you to add, edit and delete text messages. The sidebar contains a list of contacts that can be filtered using the search input. Plus, you can spice up your messages with the emoji picker!

Nom Nom: Memory Game

This ridiculously sweet memory card game was built using HTML, CSS and JavaScript. The objective is to match two cards with the same tasty treat. Incorrect guesses will cause your blood sugar to drop, so choose wisely! Scoring is based on the user's final health rating.

Pixel Art Maker

Create your very own 8-bit masterpiece with Pixel Art Maker! To get started, adjust the canvas size using the range sliders and use the color pickers to change the grid and pixel colors. When you have reached pixel perfection, click the download button and share with all your friends!

Dine Wise

Dine Wise integrates the Mapbox API to display restaurant locations on an interactive map. You can also filter restaurants by location and/or cuisine. Dine Wise uses a Service Worker to cache all of the site’s assets, so that any page that has viewed will be accessible offline.
