about

Hey there, my name is Matt and I’m a Developer and Designer from Chicago. I began my career unknowingly when I discovered Microsoft Paint on Windows 3.0. Once I mastered the spray can, I knew I was ready to take my love of art and growing interest in technology to the next level. My ambition and commitment to lifelong learning has helped me succeed in a multitude of roles throughout my career including graphic designer, art director, developer and project manager. I have always had a passion for building things and solving problems, and I am constantly discovering new ways to bring my ideas to life. When I’m not coding or designing, I enjoy playing music and spending time with my family. If you have a new project that you would like to discuss, my contact details can be found below.

work

The Loop: City Guide

Check out The Loop for the best restaurants, bars, coffee houses and attractions in the windy city! This single page React application fetches venue recommendations from the Foursquare API and integrates Google Maps to display each venue's location.

Chatbox

ChatBox is a simple chat application built with React and Redux that allows you to add, edit and delete text messages. The sidebar contains a list of contacts that can be filtered using the search input. Plus, you can spice up your text messages with the emoji picker!

Nom Nom: Memory Game

This rediculously sweet memory card game was built using HTML, CSS and JavaScript. The objective is to match two cards with the same tasty treat. Incorrect guesses will cause your blood sugar to drop, so choose wisely! Scoring is based on the user's final health rating.

Pixel Art Maker

Create your very own 8-bit masterpiece with Pixel Art Maker! Adjust the canvas size using the range sliders and use the color pickers to change the grid and pixel colors. When you have reached pixel perfection, click the download button and share with all your friends!

Mad Dash!

Mad Dash! is an HTML canvas game based on the classic arcade game Frogger. The objective is to cross the road and reach the water without colliding with any evil, man eating bugs. Complete all 15 levels to save the princess and restore peace to her kingdom!

Dine Wise

Dine Wise integrates the Mapbox API to display restaurant locations on an interactive map. You can also filter restaurants by location and/or cuisine. Dine Wise uses a Service Worker to cache all of the site’s assets, so that any page that has viewed will be accessible offline.
